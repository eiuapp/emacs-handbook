eaf proxy

当初次运行 ~M-x eaf-open-brower www.baidu.com~ 时会，因eaf的默认配置中网络设置了 socks5 127.0.0.1 1080 代理问题而找不开。 如果不要代理，则注释掉：

./lazycat-emacs/site-lisp/config/init-eaf.el 第90行左右

;; You need configuration your own local proxy program first.
;; (setq eaf-proxy-type "socks5")
;; (setq eaf-proxy-host "127.0.0.1")
;; (setq eaf-proxy-port "1080")
(setq eaf-proxy-type "")
(setq eaf-proxy-host "")
(setq eaf-proxy-port "")
