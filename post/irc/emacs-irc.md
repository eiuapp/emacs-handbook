
使用 emacs 连接 irc

irc.freenode.net

## register

https://freenode.net/kb/answer/registration

```
/msg NickServ REGISTER hello1123 273412935@qq.com
```

然后在 273412935@qq.com 中会收到邮件, 邮件会有命令, irc 中运行一下:

```
/msg NickServ VERIFY REGISTER helloit axayvhiwejea
```
